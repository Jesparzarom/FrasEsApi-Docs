Request

https://frasesapi.vercel.app/v1/frases/aleatoria/?autor=<str:autor>&categoria=<str:categoria>

Ejemplo de Response

Devuelve un objeto de frase totalmente aleatoria de determinado autor en una categoria específica. Por ejemplo autor=guido+van+rossum&categoria=programacion

{
    "frase": {
        "autor": "Guido van Rossum",
        "categoria": "programacion",
        "frase": "En la programación, como en la poesía, la belleza está en los ojos del programador.",
        "key": "guidoprog01"
    }

}
