Request

https://frasesapi.vercel.app/v1/frases/aleatoria/?categoria=<str:categoria>

Ejemplo de Response

Devuelve un objeto de frase totalmente aleatoria de determinada categoria. Por ejemplo categoria="sabiduria"

{
    "frase": {
        "autor": "Confucio",
        "categoria": "sabiduria",
        "frase": "La verdadera sabiduría está en reconocer la propia ignorancia.",
        "key": "confuciowisdom04"
    }

}
